Компиляция проекта.
В качестве компилятора был выбран компилятор G++(хотя подойдёт любой компилятор С++). В терминале переходим в папку с исходным кодом и копируем следующую команду g++ main.cpp, после выполнения должен появиться файл a.out в Linux (в Windows a.exe).

Запуск проекта.
В терминале переходим в папку с исполняемым фалом a.out и файл с исходными данными. Например:  ./a.out primer и наслаждаемся работой алгоритма.

Входные данные.
В каждом архиве находиться файлы с примерами (primer,primer1,...). В данном файле, для демонстрации работы алгоритма, представлена матрица. Формат хранения очень прост первая цифра количество строк в матрице вторая цифра количество столбцов далее элементы матрицы. В качестве разделителей можно использовать другие символы за исключением цифр(0...9), точки(.) и знака минус(-).
